# Changelog

All notable changes to Cube'n Tube are documented in this file.

## 1.1.0

### New
- Added a new **Dust Eraser** mode in Cube'n Tube.
- Added a dust size-limit slider with live preview.

- Added a real-time threshold slider for the **custom eraser**.

## 1.0.1

### Changed
- Unified right-mouse behavior and panel interaction for Cube'n Tube.
- Improved toolbar behavior so shape selection is not changed unexpectedly.

## 1.0.0

### Added
- First official release of Cube'n Tube.
